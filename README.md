# 🧠 AI Research Diagram Studio

An AI-powered web application that converts natural language prompts into publication-ready academic diagrams using IBM Granite AI and LaTeX.

## 🚀 IBM SkillBuild Internship Project

**Problem Statement #26**
**AI-Powered LaTeX Diagram Generator for Academic Research**

---

## 📌 Overview

AI Research Diagram Studio enables researchers, students, and educators to generate high-quality academic diagrams from simple text prompts.

The application combines IBM Granite AI with LaTeX (TikZ) to automatically create professional diagrams suitable for research papers and presentations.

---

## ✨ Features

- 🤖 AI-powered prompt understanding using IBM Granite AI
- 📄 Automatic LaTeX (TikZ) code generation
- 🖼️ High-quality PDF diagram generation
- ⚡ Fast and responsive React frontend
- 🔥 Flask backend API
- 📚 Academic diagram generation
- 🎯 Publication-ready output

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/BhaveshBaviskar-IT/ai-research-diagram-studio.git
```

### Backend

```bash
cd backend

pip install -r requirements.txt

python app.py
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
